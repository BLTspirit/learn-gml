# Welcome

This is the first lesson of the course. By the end of it, you will have written and run your first piece of code.

## What is GameMaker?

GameMaker is a tool for making 2D games. It includes a programming language called GML, which is what this course will teach you.

## What is code?

Code is a set of instructions that tells a computer what to do.

## Recap

- GameMaker is a tool for making games.
- GML is the language we use to write instructions for it.
